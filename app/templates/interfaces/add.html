Tips
Interface Naming
  • Cisco: GigabitEthernet0/1, FastEthernet0/0
  • Palo Alto: ethernet1/1, ethernet1/2
  • FortiGate: wan1, lan, port1
IP Address Format
  • IPv4: Include CIDR notation (e.g., 10.0.0.1/24)
  • IPv6: Include prefix length (e.g., 2001:db8::1/64)
  • Leave blank if interface has no IP address
When to Add Manually
  • Interface not detected by automatic collection
  • Virtual or sub-interfaces
  • Temporary documentation needs
  • Planning for future configurations
